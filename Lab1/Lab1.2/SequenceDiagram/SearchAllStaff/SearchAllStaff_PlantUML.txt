@startuml SearchAllStaff
actor Admin as User
participant System
participant Server

activate User
User -> System: Access "Staff List" screen
activate System
User -> System: Search for any Staff
System -> Server: Send "Search" request
activate Server
Server -> Server: Find staff
alt Staff exist
    Server --> System: List of Staff
    System --> User: Show List on screen
else Staff do not exist
    Server --> System: No staff found
    deactivate Server
    System --> User: Show empty List on screen
    deactivate System
end 
deactivate User

@enduml
