@startuml ViewDocumentInformation
actor Admin_Staff_User as User
participant System
participant Server

activate User
User -> System: Access "Document Information" screen
activate System
System -> Server: Send "View document information" request
activate Server
Server -> Server: Prepare document information
alt Valid document
    Server --> System: Request Successful
    Server --> System: Document information
    System --> User: Show document information on screen
else Invalid document
    Server --> System: Request failed
    deactivate Server
    System --> User: Show error notification
    deactivate System
end 
deactivate User

@enduml
