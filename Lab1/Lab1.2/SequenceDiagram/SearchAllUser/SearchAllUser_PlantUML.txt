 @startuml SearchAllUsers
actor Admin as User
participant System
participant Server

activate User
User -> System: Access "User List" screen
activate System
User -> System: Search for any User
System -> Server: Send "Search" request
activate Server
Server -> Server: Find user
alt User(s) exist
    Server --> System: List of User(s)
    System --> User: Show List on screen
else User(s) do not exist
    Server --> System: No user found
    deactivate Server
    System --> User: Show empty List on screen
    deactivate System
end 
deactivate User

@enduml
